## 🚀 如何使用

### 方式 1: Docker（推荐）

\`\`\`bash
# 克隆项目
git clone <repo-url>
cd ai-agent-platform

# 启动
docker-compose up -d

# 访问
open http://localhost:3000
\`\`\`

### 方式 2: 本地开发

\`\`\`bash
# 后端
npm install
cp .env.example .env
npm run dev

# 前端
cd frontend
npm install
npm run dev
\`\`\`
