## 🎯 里程碑

### Milestone 1: 最小可用版本 (MVP)
**目标：** 能对接 1 个业务系统，处理简单查询
**时间：** 4-6 周
**标志：** 
- 可以通过自然语言查询业务数据
- 至少有 3 个可用 Skills
- 有简单的 Web 界面

### Milestone 2: 功能增强
**目标：** 支持多系统、多模型
**时间：** 6-8 周
**标志：**
- 对接至少 3 个业务系统
- 支持至少 3 个大模型
- 会议助手基础功能
- 动态页面生成

### Milestone 3: 自主进化
**目标：** AI 能自主学习新系统
**时间：** 持续
**标志：**
- Skill 自动生成
- 系统自发现
- Skill 市场
