# 适配向导前端设计

## 🎨 UI 设计理念

**设计原则：**
- 简洁明了，每一步只做一件事
- 实时反馈，让用户知道进度
- 智能提示，AI 辅助用户完成配置
- 可视化，展示 AI 分析结果

## 💻 技术实现

### 前端技术栈
- **框架**: React 18 / Vue 3
- **UI 组件**: Ant Design / Element Plus
- **状态管理**: Zustand / Pinia
- **代码编辑器**: Monaco Editor
- **可视化**: ECharts

### API 调用示例

\`\`\`typescript
// 创建会话
const response = await fetch('/api/wizard/start', {
  method: 'POST',
  body: JSON.stringify({ userId: 'user123' })
});

// 步骤 1: 保存系统信息
await fetch(\`/api/wizard/\${sessionId}/system-info\`, {
  method: 'POST',
  body: JSON.stringify({
    name: '自然 ERP',
    type: 'erp',
    description: '企业资源管理系统'
  })
});

// 步骤 2: 保存 API 配置
await fetch(\`/api/wizard/\${sessionId}/api-config\`, {
  method: 'POST',
  body: JSON.stringify({
    baseUrl: 'https://api.natural.com/v1',
    authType: 'api_key',
    apiKey: 'sk-xxx'
  })
});

// 步骤 4: AI 分析
await fetch(\`/api/wizard/\${sessionId}/analyze\`, {
  method: 'POST'
});

// 步骤 7: 部署
await fetch(\`/api/wizard/\${sessionId}/deploy\`, {
  method: 'POST'
});
\`\`\`
